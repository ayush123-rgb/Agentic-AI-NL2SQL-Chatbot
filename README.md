# Agentic AI NL2SQL Chatbot

An Agentic AI-powered Natural Language to SQL (NL2SQL) chatbot that converts user questions into SQL queries, executes them on a relational database, and generates human-readable responses.

The system supports conversational context, session-based memory, SQL validation, semantic query reuse, and automated session management.


## Features

### Natural Language to SQL Conversion

Convert plain English questions into executable SQL queries.

**Example:**

Input:

```
Show all payment values
```

Generated SQL:

```
SELECT payment_value
FROM payments;
```


### Conversational Context Management

The chatbot remembers context within a session.

Example:

User:

```
Show all payment values
```

Follow-up:

```
Show average value
```

The chatbot understands that "value" refers to payment values.


### Session-Based Memory

Each conversation is associated with a unique session ID.

Sessions remain isolated from one another, preventing context leakage across users.


### UUID-Based Session Generation

Automatically generates unique session IDs.

Endpoint:

```
POST /create-session
```

Response:

```
{
  "session_id": "f4f5c5de-7d0c-4f39-88d8-2d4c96d5e7aa"
}
```


### SQL Query Validation

Prevents execution of unsafe SQL statements.

Blocked operations include:

* DROP
* DELETE
* UPDATE
* ALTER
* TRUNCATE

Only safe read operations are allowed.


### Semantic Query Memory

The system stores:

* User Input
* Generated SQL
* Chatbot Response
* Session ID

Before generating a new SQL query, the chatbot performs semantic similarity search on previously stored queries.

Example:

Stored Query:

```
Show all payment values
```

New Query:

```
Display payment amounts
```

If the meaning is similar, the chatbot reuses the stored result instead of generating a new SQL query.

This reduces:

* LLM API calls
* Response latency
* Computational cost


### Query History Storage

Every successful interaction is stored in the database.

Stored information:

| Column          | Description                |
| --------------- | -------------------------- |
| session_uid     | Unique session identifier  |
| user_input      | Original user query        |
| generated_sql   | SQL generated by the model |
| output_response | Final chatbot response     |
| created_at      | Timestamp                  |

## Installation

### Clone Repository

```
git clone <repository-url>
cd project
```

### Create Virtual Environment

```
python -m venv venv
```

### Activate Virtual Environment

Windows:

```
venv\Scripts\activate
```

Linux / Mac:

```
source venv/bin/activate
```

### Install Dependencies

```
pip install -r requirements.txt
```

Additional dependencies:

```
pip install sentence-transformers
pip install scikit-learn
```
